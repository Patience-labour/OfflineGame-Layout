## ü¶ñ Project Overview

This project is an independent implementation of the famous Google Chrome "T-Rex Runner" offline game. The goal is to recreate the basic mechanics, physics, and feel of the original game using only **HTML5, CSS3, and Vanilla JavaScript**, without game engines.

**Primary Objective**: Master the fundamentals of game loops, state management, collision detection, and smooth animation in a web browser.
